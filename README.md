# GetYourTicket

open terminal in the current folder
1. Create a virtual environment using your preferred method 
$ python3 -m venv env
2. Activate the virtual environment.
$ source env/bin/activate
3. Install the required packages by running pip install -r requirements.txt in your terminal.
4. Run python setup.py to set up the database.
5. Run python app.py to start the Flask server.
6. Go to http://127.0.0.1:5000 in your web browser to access the web application.
7. Navigate to ‘Register’ and register as Admin using ‘admin123’ as username and password of your choice.(all other username will be Users not Admin)
